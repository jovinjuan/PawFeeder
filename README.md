 # Paw Feeder 🐈 🐕
Paw Feeder is an IoT based feeding systems integrated with mobile apps designed for busy owners who works and travels a lot with a limited time to take care of their pets.

- Jovin Juanlie

⚙️ Core Features
- Automate feeding based on feeding schedules
- Monitor stock (food & drink) level, pet condition (through ESP32 cam module) and daily feeding logs
- Gamified feeding process by leveling system

💻 Tech Stacks and Tools

- Firebase 
- Arduino C
- Android Studio
